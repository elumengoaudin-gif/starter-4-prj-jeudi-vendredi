## Bugs corrigés (jeudi)

1. **Bug** : Le CSS ne se chargeait pas
   - **Cause** : Chemin incorrect dans index.html
   - **Fix** : Corrigé le chemin vers styles.css

2. **Bug** : Images ne s'affichaient pas
   - **Cause** : Source image incorrecte
   - **Fix** : Corrigé les chemins vers assets/

3. **Bug** : Classe HTML non stylée
   - **Cause** : Manquait le point (.) devant la classe en CSS
   - **Fix** : Ajouté le sélecteur correct

4. **Bug** : Faute d'orthographe dans classe HTML
   - **Fix** : Corrigé le nom de la classe

5. **Bug** : Faute dans propriété CSS
   - **Cause** : font-weight mal écrit
   - **Fix** : Corrigé l'orthographe

6. **Bug** : justify-content ne fonctionnait pas
   - **Cause** : Manquait display: flex
   - **Fix** : Ajouté display: flex au conteneur
